# Resume Screening System

## Overview

The Resume Screening System is a Machine Learning project designed to assist recruiters in evaluating candidate profiles efficiently. The system analyzes candidate information such as skills, education, experience, certifications, salary expectations, project counts, and AI assessment scores to predict suitable job roles.

## Problem Statement

Organizations often receive a large number of job applications, making manual resume screening time-consuming and challenging. Recruiters must evaluate multiple factors before determining whether a candidate is suitable for a specific position. This project aims to automate the initial screening process by using machine learning techniques to analyze candidate profiles and classify them into appropriate job roles.

## Dataset

The dataset contains 1000 candidate records with the following attributes:

* Resume ID
* Name
* Skills
* Experience (Years)
* Education
* Certifications
* Job Role
* Recruiter Decision
* Salary Expectation ($)
* Projects Count
* AI Score (0–100)

## Project Objectives

* Clean and preprocess candidate data.
* Handle missing values and duplicate records.
* Perform feature engineering.
* Convert textual data into machine-readable format.
* Conduct exploratory data analysis (EDA).
* Train machine learning models for job role prediction.
* Evaluate and compare model performance.
